---
title: abs
description: returns the absolute value of a number.
version: 1.9.1
works: true
---
### Example 1
input: {% raw %}
<code>
{{ -17 | abs }}
</code>
{% endraw %}
output:
<code>
{{ -17 | abs }}
</code>

### Example 2
input: {% raw %}
<code>
{{ 4 | abs }}
</code>
{% endraw %}
output:
<code>
{{ 4 | abs }}
</code>

### Example 3
input: {% raw %}
<code>
{{ "-19.86" | abs }}
</code>
{% endraw %}
output:
<code>
{{ "-19.86" | abs }}
</code>
