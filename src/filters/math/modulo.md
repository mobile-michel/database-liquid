---
title: modulo
description: returns the remainder of a division operation.
version: 1.9.1
works: true
---
### Example 1
input: {% raw %}
<code>
{{ 3 | modulo: 2 }}
</code>
{% endraw %}
output:
<code>
{{ 3 | modulo: 2 }}
</code>

### Example 2
input: {% raw %}
<code>
{{ 24 | modulo: 7 }}
</code>
{% endraw %}
output:
<code>
{{ 24 | modulo: 7 }}
</code>

### Example 3
input: {% raw %}
<code>
{{ 183.357 | modulo: 12 }}
</code>
{% endraw %}

<code>
{{ 183.357 | modulo: 12 }}
</code>
